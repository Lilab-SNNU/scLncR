## Quick Start Guide

### Step 1: Configure Parameters
Navigate through each module tab and configure the parameters according to your data and analysis needs.

### Step 2: Save Configuration
Click "Save Config" to save your settings for future use.

### Step 3: Run Pipeline
Go to the "Run Pipeline" tab, select which modules to run, and click "Run Pipeline".

### Step 4: Monitor Progress
Watch the output log for real-time updates on pipeline execution.

### Step 5: View Results
Results will be saved in the specified output directories.
